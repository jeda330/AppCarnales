# AppCarnales 🍖🍷
*Aplicación móvil para gestión de licoreras*

## Funcionalidades
- Registro de ventas
- Sistema de puntos gamificado
- Gestión de niveles de usuarios

## Tecnologías
- Android Studio (Kotlin/Java)
- Firebase/MySQL (backend)

## Cómo Contribuir
1. Clona el repositorio
2. Abre en Android Studio
